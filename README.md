# RTO-Management-System
Create a database rto in phpmyadmin and import rto.sql from database folder.


-> To get qr code image download composer and add its path to environment variable of your PC.

How to run this project:
1.Install Xampp.
2.locate htdocs folder inside xampp folder.
3.add all the files to htdocs folder.
4.start apache and MySql server.
5.open localhost/phpmyadmin in browser.
6.run sql query to create a database.
7.open localhost/(your_file_name) in browser.
8.This project will run on localhost server.
